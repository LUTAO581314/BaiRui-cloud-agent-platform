# Third-Party Notices

## BaiLongma Brain UI

The `bairui-agent` user workspace adapts the panel structure, chat console
interaction model, and visual language of BaiLongma Brain UI.

- Upstream: https://github.com/xiaoyuanda666-ship-it/BaiLongma
- Reviewed commit: `0e243bc518cebbdb74f114e98b1189133abbda63`
- Upstream paths: `brain-ui.html`, `src/ui/brain-ui/app-shell.js`,
  `src/ui/brain-ui/chat.js`, and `src/ui/brain-ui/styles.css`
- License: MIT

The Bairui adaptation retains the BaiLongma presentation layer only. It uses
Bairui authentication, organization isolation, conversation APIs, storage,
and Hermes Runtime Boundary instead of BaiLongma local runtime APIs.
